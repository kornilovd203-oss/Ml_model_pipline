# Ml_model_pipline
Цель данного проекта — создать модель машинного обучения, способную предсказать, в какой день недели пользователь совершил коммит по учебному проекту, зная проект, который выполнял конкретный пользователь, час суток и колличесвто попыток сдачи проекта. 

Целевая переменная — день недели, поэтому я использую модель многоклассовой классификации.

Перед работой над моделью данные были подготовлены. 

В данном проекте создаю пайплайн первичной обработки данных, далее обучая несколько моделей машинного обучения и сохраняю лучший результат.
